Informações Adicionais

Para a opção de "Formatação de Data, entenda o padrão:
- dd-mm-yyyy = dia/mês/ano, exemplo: 20/11/2003;
- mm-dd-yyyy = mês/dia/ano, exemplo: 11/20/2003;
- yyyy-mm-dd = ano/mês/dia, exemplo: 2003/11/20;
- yyyy-dd-mm = ano/dia/mês, exemplo: 2003/20/11.
